Step 1: Adding the Javascript

The FXWebApp code uses two Javascript files.
  1. dtjava.js which is part of the Java Deployment Toolkit
  2. fxdeploy-common.js which is part of the FXDeployment code
Use of fxdeploy-common.js is optional: you can directly call the functions supplied in dtjava.js, but fxdeploy-common.js makes this easier.
In the description that follows, it is assumed that fxdeploy-common.js is being used. Add the <script> declarations to the <head> element of your HTML file:
<script type="text/javascript" src="http://java.com/js/dtjava">
<script type="text/javascript" src="pathto/fxdeploy-common.js">
Note that you need to set the path to locate the fxdeploy.common.js file on your site.

Step 2: Adding the required HTML

  • fxml-placeholder if you are using an FXML file
